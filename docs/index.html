Browser-Erweiterung für Firefox und Chrome

Klartext statt Politikgelaber.

Klartext ist die Browser-Erweiterung, die politische Aussagen in verständliche Sprache übersetzt.

Schluss mit Politiksprech: Klartext macht aus weichgespülten Politikerphrasen das, was wirklich gemeint ist.

Zu Firefox Add-ons Zum Chrome Web Store

Was es macht

Klartext macht politische Sprache lesbarer.

Verständlich formuliert

Politische Aussagen werden in eine deutlich direktere Sprache übersetzt.

Direkt auf der Seite

Klartext arbeitet dort, wo du liest: direkt auf Nachrichten- und Politikseiten im Browser.

Schnell ein- und ausschaltbar

Auf der Seite erscheint eine kleine Leiste, über die du Klartext jederzeit aktivieren oder deaktivieren kannst.

So nutzt du Klartext

In wenigen Schritten startklar.

  1. Installiere Klartext über Firefox Add-ons oder den Chrome Web Store.
  2. Öffne eine politische Seite oder einen entsprechenden Artikel im Browser.
  3. Klartext übersetzt Formulierungen direkt beim Lesen in deutlichere Sprache.
  4. Wenn du möchtest, schaltest du die Funktion über die Leiste auf der Seite wieder aus.

Datenschutz

Knapp und klar.

Laut den Angaben in Firefox Add-ons und im Chrome Web Store erhebt oder nutzt Klartext keine Daten von dir.

Die Erweiterung braucht Zugriff auf Webseiten, damit sie Texte direkt in deinem Browser in verständlichere Sprache übersetzen kann.
